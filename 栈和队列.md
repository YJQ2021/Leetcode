leetcode栈和队列
===
1、用队列模拟栈（225 easy）
---
用两个队列模拟栈，关键在于入队。<br>
```
public void push(int x) {
        q1.offer(x);
        while(!q2.isEmpty())
            q1.offer(q2.poll());
        Queue temp = q1;
        q1 = q2;
        q2 = temp;
    }
 ```
 可保证q2的队头总是指向最后一个入栈的元素（即栈顶）。这样操作完后，q2可以直接当作栈来使用，栈顶就是q2的队首。<br>
